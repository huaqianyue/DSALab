欢迎使用 DSALab

面向数据结构与算法学习的学习平台

题目导入与管理

云端导入和合并您的算法题目

  • 支持 JSON 格式题目导入
  • 本地存储，支持离线使用
  • 智能过滤，确保题目准确
  • 自动合并本地和远程题库
如何使用：点击顶部工具栏的 "导入" 按钮，选择 JSON 题目文件

题目描述与导航

清晰展示题目内容，快速切换题目

  • 富文本显示，支持代码高亮
  • 智能解析函数签名和测试示例
  • 便捷的上一题/下一题导航
  • 与编辑器状态实时同步
如何使用：点击左侧边栏的 "题目列表" 选择题目，或在题目描述页面使用导航按钮

音频录制与讲解

录制您的解题思路和讲解过程

  • 高质量音频录制
  • 支持暂停、继续录制
  • 实时回放和进度控制
  • 自动保存到题目工作区
如何使用：在题目描述页面底部的音频面板中，点击 "录制" 按钮开始录音

编译运行系统

支持 C++ 编译和运行环境

  • 支持 C++20 标准
  • 一键编译运行
  • 详细的错误诊断信息
  • 集成控制台输出显示
如何使用：使用顶部工具栏的 "编译" 或 "运行" 按钮

可视化调试

专业的调试工具，帮助您排查程序问题

  • 断点管理和可视化
  • 实时变量监控
  • 单步执行和步进控制
  • 表达式求值
如何使用：在编辑器中设置断点，然后点击底部工具栏的 "调试" 开始调试

自动测试与评分

智能测试您的代码，自动给出分数

  • 自动提取和测试函数
  • 实时评分和通过率
  • 测试结果历史记录
  • 详细的测试报告
如何使用：编写完代码后，点击顶部工具栏的 "测试" 按钮运行自动测试

工作成果导出

导出您的编程作业和学习成果

  • 打包为 ZIP 文件
  • 包含测试分数信息
  • 选择性导出完成的题目
  • 包含学习历史记录
如何使用：点击顶部工具栏的 "导出" 按钮，选择要导出的题目

编译器设置

灵活配置编译环境和选项

  • 编译标准和优化级别
  • MinGW 路径管理
  • 编码和警告控制
  • 编译字符设置
如何使用：点击菜单栏 "工具" → "编译设置" 进行配置。如遇中文显示问题，可尝试设置编译字符。

快速开始指南

1

导入题目

云端自动导入或点击 "导入" 按钮，选择包含算法题目的 JSON 文件

2

选择题目

在左侧 "题目列表" 中选择要解答的题目

3

编写代码

在编辑器中编写 C++ 代码，使用 "编译" 和 "运行" 测试

4

录制讲解

在题目描述页面底部录制您的解题思路和讲解

5

测试和导出

使用 "测试" 验证代码，最后 "导出" 您的成果

提示：您可以随时通过菜单栏 "帮助" → "欢迎使用 DSALab" 重新打开此欢迎页面
